# test-super-linter

- ### Requeriments
  - NodeJS
    - ESLint (globally or in the project)
  - Visual Studio Code
  - Java 8+

- ### Clone repository
      git clone https://github.com/fidevs/test-super-linter.git
    #### Or SSH
      git clone git@github.com:fidevs/test-super-linter.git
      cd test-super-linter
